# Lead Magnet Formats

Complete guide to choosing and creating the right format.

---

## Quick Reference

| Format | Best For | Effort | Perceived Value |
|--------|----------|--------|-----------------|
| Checklist | Process followers | Low | Medium |
| Template | DIYers | Medium | High |
| Swipe File | Creatives | Medium | High |
| Guide/Ebook | Deep learners | High | Medium-High |
| Cheat Sheet | Quick reference | Low | Medium |
| Calculator | Decision makers | High | Very High |
| Assessment/Quiz | Self-discovery | High | Very High |
| Mini-Course | Comprehensive learners | Very High | Very High |
| Resource Library | Ongoing reference | Medium | High |
| Toolkit | Practitioners | Medium | High |

## Format Deep Dives

### Checklist

**What it is:** Step-by-step list to complete a process or ensure nothing is missed.

**Best for:**
- Process-oriented tasks
- Quality assurance
- Repeatable workflows
- Launch/project preparation

**Structure:**
```
Title: [Process] Checklist
□ Step/Item 1 - [brief context if needed]
□ Step/Item 2
□ Step/Item 3
...
[Optional: Pro tip box]
```

**Examples:**
- "Product Launch Checklist: 47 Things Before You Go Live"
- "SEO Audit Checklist: 23 Points to Fix Today"
- "Sales Call Prep Checklist"

**Time to create:** 2-4 hours
**Ideal length:** 1-3 pages

---

### Template

**What it is:** Pre-built framework they can customize for their situation.

**Best for:**
- People who want shortcuts
- Repetitive tasks
- Proven structures
- Time-starved audiences

**Structure:**
```
Title: [Type] Template
Instructions: How to use this template
[The Template Itself]
[Example of filled-in version]
[Tips for customization]
```

**Examples:**
- "Cold Email Template That Gets 40% Response Rates"
- "Content Calendar Template (Google Sheets)"
- "Sales Page Template: Fill-in-the-Blank"

**Time to create:** 4-8 hours
**Format:** Google Docs, Sheets, Notion, Figma

---

### Swipe File

**What it is:** Collection of proven examples to model or draw inspiration from.

**Best for:**
- Creatives needing inspiration
- Copywriters
- Designers
- Anyone creating similar assets

**Structure:**
```
Title: [X] [Type] Swipes
Introduction: How to use these examples
[Example 1] + Analysis of why it works
[Example 2] + Analysis
...
[Tips for adapting]
```

**Examples:**
- "50 High-Converting Headlines (Swipe File)"
- "Email Subject Lines That Got 50%+ Open Rates"
- "Landing Page Designs That Convert"

**Time to create:** 4-8 hours
**Ideal size:** 10-50 examples

---

### Guide/Ebook

**What it is:** Comprehensive resource teaching a topic in depth.

**Best for:**
- Complex topics needing explanation
- Establishing authority
- SEO/content upgrades
- Evergreen education

**Structure:**
```
Title Page
Table of Contents
Introduction (the promise)
Chapter 1: [Foundation]
Chapter 2: [Core method]
Chapter 3: [Advanced tactics]
Chapter 4: [Implementation]
Conclusion + Next Steps
About/CTA
```

**Examples:**
- "The Complete Guide to LinkedIn Outreach"
- "B2B Content Strategy: From Zero to Pipeline"
- "Pricing Your Services: A Framework"

**Time to create:** 8-20 hours
**Ideal length:** 15-40 pages (not 100+ page monsters)

---

### Cheat Sheet

**What it is:** Quick-reference condensed information.

**Best for:**
- Complex info that needs simplification
- Frequent reference
- Technical topics
- Beginners needing shortcuts

**Structure:**
```
Title: [Topic] Cheat Sheet
[Section 1: Key concepts]
[Section 2: Common formulas/patterns]
[Section 3: Quick reference table]
[Section 4: Common mistakes]
```

**Examples:**
- "Google Analytics 4 Cheat Sheet"
- "Email Marketing Metrics: What Good Looks Like"
- "Copywriting Formulas Cheat Sheet"

**Time to create:** 1-2 hours
**Ideal length:** 1-2 pages, single printable sheet

---

### Calculator/Tool

**What it is:** Interactive tool that produces personalized output.

**Best for:**
- Data-driven decisions
- ROI calculations
- Assessments
- Personalized recommendations

**Structure:**
```
Input fields (what they enter)
Calculation logic
Output (their personalized result)
Interpretation guide
Next steps based on result
```

**Examples:**
- "Pricing Calculator: What to Charge for Your Services"
- "ROI Calculator: Is This Tool Worth It?"
- "Content Volume Calculator: How Much to Publish"

**Time to create:** 8-16 hours
**Format:** Google Sheets, custom web tool, Typeform

---

### Assessment/Quiz

**What it is:** Self-evaluation that reveals something about them.

**Best for:**
- Self-discovery
- Segmentation
- Engagement
- Personalized recommendations

**Structure:**
```
Introduction (what they'll learn about themselves)
Questions (5-20)
Scoring logic
Results (3-5 possible outcomes)
Personalized next steps per outcome
```

**Examples:**
- "What's Your Marketing Personality?"
- "Content Strategy Assessment: Where Are Your Gaps?"
- "Sales Process Maturity Quiz"

**Time to create:** 8-20 hours
**Format:** Typeform, ScoreApp, custom

---

### Mini-Course

**What it is:** Multi-part educational sequence delivered over time.

**Best for:**
- Complex topics
- Building relationship
- Establishing expertise
- High-ticket lead gen

**Structure:**
```
Day 1: [Foundation + quick win]
Day 2: [Core concept #1]
Day 3: [Core concept #2]
Day 4: [Implementation]
Day 5: [Next steps + offer]
```

**Examples:**
- "5-Day Email Marketing Bootcamp"
- "Learn SEO in 7 Days"
- "Sales Confidence Mini-Course"

**Time to create:** 10-30 hours
**Format:** Email sequence, video series, or both

---

### Resource Library

**What it is:** Collection of multiple resources in one place.

**Best for:**
- Comprehensive solutions
- Ongoing reference
- High perceived value
- Multiple audience segments

**Structure:**
```
Welcome/Navigation guide
Category 1: [Resources]
Category 2: [Resources]
Category 3: [Resources]
How to use / Getting started guide
```

**Examples:**
- "The Marketing Toolkit: 25+ Resources"
- "Startup Resource Library"
- "Content Creator's Vault"

**Time to create:** Varies (often curated)
**Format:** Notion, members area, Drive folder

## Format Selection Decision Tree

```
START: What does your audience need most?

├─ Quick answer to specific question?
│  └─ CHEAT SHEET or CHECKLIST
│
├─ Help doing something themselves?
│  └─ Is it creative or process-based?
│     ├─ Creative → TEMPLATE or SWIPE FILE
│     └─ Process → CHECKLIST or TEMPLATE
│
├─ Making a decision?
│  └─ Is data involved?
│     ├─ Yes → CALCULATOR or ASSESSMENT
│     └─ No → GUIDE or ASSESSMENT
│
├─ Learning something comprehensively?
│  └─ Do they have time to consume?
│     ├─ Yes → GUIDE or MINI-COURSE
│     └─ No → CHEAT SHEET or CHECKLIST
│
└─ Ongoing reference/support?
   └─ RESOURCE LIBRARY or TOOLKIT
```

## Format Don'ts

**Don't:**
- Create a 100-page ebook (nobody reads it)
- Over-design simple formats
- Use PDFs when interactive would be better
- Gate something freely available elsewhere
- Choose format before understanding audience need
- Spend 40 hours on something you haven't validated
